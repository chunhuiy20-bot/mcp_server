│                                                                              │
│  贝叶斯公式:                                                                 │
│    P(录取|特征) = P(特征|录取) × P(录取) / P(特征)                           │

│  使用似然比和先验比值 (Odds) 形式:                                           │

│    后验比值 = 似然比 × 先验比值                                              │
│    最终概率 = 后验比值 / (1 + 后验比值)                                      │

│     将基于特征的概率与真实录取率结合                                         │

                              核心设计理念
═══════════════════════════════════════════════════════════════════════════════

1️⃣  细粒度量化
   └─ 将模糊的申请要素转化为可计算的数值指标

2️⃣  非线性评分
   └─ 高分段边际效用递减 (GPA 3.95→4.0 提升有限)

3️⃣  动态权重
   └─ 根据大学层级和竞争环境自动调整各维度重要性

4️⃣  学校背景校准
   └─ 不同学校的GPA可信度不同,只能向下调整

5️⃣  竞争感知
   └─ 学霸扎堆时,学术优势被削弱,其他维度更重要

6️⃣  贝叶斯融合
   └─ 结合特征评分和真实录取率,避免过度乐观

7️⃣  特殊情况处理
   └─ 世界级特长可以突破常规逻辑

═══════════════════════════════════════════════════════════════════════════════
                              数学公式总结
═══════════════════════════════════════════════════════════════════════════════

调整后GPA = min(原始GPA × 学校信任系数, 原始GPA)

综合分 = Σ(维度分数ᵢ × 调整后权重ᵢ)

基础概率 = sigmoid(综合分) = 1 / (1 + e^(-(综合分-中心)/标准差))

后验概率 = (基础概率 × 先验概率) / 
           (基础概率 × 先验概率 + (1-基础概率) × (1-先验概率))

最终概率 = 后验概率 × 特殊才能加成系数
